Field Description
digit If your data should only have numbers. Example column - Reporting Year
date If you want to check if date is in valid date format: MM/DD/YYYY or M/D/YY
decimal This check if your format is a number with 2 digits after the decimal point.
phone Checks if the phone is in the correct format
zipcode Checks zipcode in two formats: a 5 digit number and a 5 digit number followed by a '-' and 4 digits.
email Checks if email is in valid format.
